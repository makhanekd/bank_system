# Bank System (Task 1)

[Url Task 1](https://junovnet.gitbook.io/programmirovanie/zadachi-po-programmirovaniyu/zadanie-bankovskaya-sistema)

## How to run

### Install

```shell
poetry install
poetry shell
python src/main.py
```

### Run

```shell
python src/main.py
```

or

```shell
python -m src.main
```

## How to test

```shell
poetyr shell
pytest
```
